The Old Terminal

This was how computers used to work. They didn't use mice and only used the keyboard. How did they get things done?

The terminal uses commands to get things done. It gives us full control over the system. Many things that you can click just use the terminal behind the scenes. It can be quite useful. When using the terminal, you use it in a directory and "work" in it. First, you type your command. Then, you give a space and type the stuff you want to give to the command. This terminal can be accessed by clicking the black rectangle at the top or going to Menu->Accessories->Terminal. An example of a command is below. 


# Lists everything in the current directory
dir
# Creates a directory named hello
mkdir hello

Our Commands

The terminal is a place where you can enter commands directly to the computer. It doesn't need fancy graphics and it is where we run our program. We can go to different parts of the computer by using the "cd" command. "python" is how we run our program.
