Booleans

So there is a new type of variable that we are introducing today! The boolean

It can have the value of true or false. Try telling if the following statements are true or false.

  • Apples grow on trees
  • 1 + 1 = 3
  • The sky is green
  • You are in San Francisco

In python, true and false can be assigned to a variable like this:


# Both create a variable meaning True
x = True
x = 1
# Both create a variable meaning False
x = False
x = 0

Notice the uppercase

You can compare things that are the same type! Use the == to compare things in Python. It will give back a boolean
